# Quantum–DNA Hybrid Image Encryption

This project implements a hybrid image encryption system combining:
- NEQR-based quantum image representation
- Chaotic key-driven quantum scrambling
- DNA encoding and diffusion
- Quantum reconstruction and evaluation (PSNR, SSIM)

## Features
- Quantum position–intensity encoding
- Chaotic Henon and logistic–sine key generation
- DNA-based diffusion
- Modular Python implementation
- Google Colab compatible

## Requirements
```bash
pip install -r requirements.txt
```
